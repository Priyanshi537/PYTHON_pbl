# Quality Prediction System using Machine Learning

## 📌 Objective
To predict the quality of a product using machine learning techniques.

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn

## 📊 Features
- Data preprocessing
- Model training
- Prediction system
- Model saving using pickle

## ▶️ How to Run
1. Install dependencies:
   pip install pandas numpy scikit-learn

2. Run:
   python main.py

## 📈 Output
- Model predicts quality based on input features
